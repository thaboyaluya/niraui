Button Group

Nira UI Button Group component allows you to group multiple buttons together, making it easier for users to toggle between related actions.

Default Button Group

Use the default .button-grouped class to create a simple button group.

Button Group Sizes

Nira UI supports multiple button group sizes so you can adapt them to your design system.







Rounded Button Groups

Create rounded button groups using the .button-grouped-rounded class for a softer UI style.

Button Group Colors

Apply color variants to your button groups for different UI states or actions.
