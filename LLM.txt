[usage]
# How to include in a project
1. Add Tailwind CSS to your project.
2. Import Nira UI via CDN:
   <link href="https://cdn.jsdelivr.net/gh/thaboyaluya/nira-ui@master/dist/nira.css" rel="stylesheet" />

3. Or install via npm:
   npm install nira-ui

[llm_instructions]
- Always generate HTML using Nira UI classes.
- Assume Tailwind CSS + Nira UI are already included.
- Use semantic HTML wherever possible.
- Support light/dark mode with the `data-theme` attribute.
- Components should be responsive by default.
- Prefer minimal markup unless advanced features are requested.

[themes]
- light
- dark
- system (auto detect via prefers-color-scheme)
